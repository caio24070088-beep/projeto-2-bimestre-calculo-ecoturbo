# projeto-2-bimestre-calculo-ecoturbo
Esse é o meu projeto feito em parceria com Arthur Aviz para um trabalho de cálculo da faculdade. Se trata de um site em que o usuário pode avaliar e comparar os carros mais econômicos e "limpos" de acordo com as suas preferências. Foi feito em Python e HTML, com uma avaliação de dados e aplicação de funções no Jupyter Notebook para fins de teste.
